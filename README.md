# 🚀 Kanban Task Manager

A modern and responsive Kanban-style task management web app built with React.  
This app helps users organize tasks visually by moving them across different stages — **To Do, In Progress, and Done**.

## 📌 Project Overview

Kanban Task Manager is a single-page application where users can:

- Create tasks
- Manage workflow visually
- Track progress easily

All data is stored in the browser using **localStorage**, so no backend is required.

---

## ✨ Features

### 📝 Task Management

- Add new tasks with title, description, and priority
- Edit existing tasks
- Delete tasks with confirmation

### 🔄 Drag & Drop

- Move tasks between:
  - To Do
  - In Progress
  - Done
- Smooth and interactive drag-and-drop experience

### 🔍 Search & Filter

- Real-time task search
- Filter tasks by priority:
  - High
  - Medium
  - Low

### 📊 Task Tracking

- Task count for each column
- Summary cards:
  - Total Tasks
  - In Progress
  - Completed

### 🌙 Dark / Light Mode

- Toggle between dark and light themes
- Preference saved in localStorage

### 🔔 Notifications

- Toast notifications for:
  - Add
  - Edit
  - Delete
- Confirmation alerts for delete actions

### 🧹 Extra Features

- Clear all completed tasks
- Empty state messages for better UX
- Priority color badges:
  - 🔴 High
  - 🟡 Medium
  - 🟢 Low
- Smooth animations for actions

## 📦 Installation & Setup

Clone the repository:

git clone https://github.com/shsobur/Kanban-Task-Board.git  
cd kanban-task-manager

Install dependencies:

npm install

Run the project:

npm run dev
